Static Semantics

Definition of static values

V ::= identifier κ ::= type | +type | -type | ( κ × κ ) | ( κ → κ ) | ( κ ∧ κ ) τ ::= bool | int | rational | string | binary | (τ×τ) | ( τ → τ ) | ( ∀ V . τ ) | ( ∃ V . τ ) Γ ::= (T,V,F) Ϝ ::= ((Vn),(τm),τ,T,V,F) T : V→τ×κ V : V→τ F : V→Ϝ τ1×⋯×τn = (τ1×(⋯×τn)) ∀ V1 , ⋯ , Vn . τ = ( ∀ V1 . ⋯ ( ∀ Vn . τ ) ) (τ0) → τ = τ (τn) → τ = τ1×⋯×τn→τ ∀ (V0) . τ = τ ∀ (Vn) . τ = ∀ V1 , ⋯ , Vn . τ

Rules for static values

κ ≤ κ κ1 ≤ κ2   κ2 ≤ κ3 κ1 ≤ κ3 type ≤ type type ≤ +type type ≤ -type

Rules for qualified-identifier

Γ = (T, V, F)   T (identifier) = ( τ , κ ) Γ ⊢ identifier ⇓ ( τ , κ ) Γ = (T, V, F)   V (identifier) = τ Γ ⊢ identifier ⇓ τ Γ = (T, V, F)   F (identifier) = Ϝ Γ ⊢ identifier ⇓ Ϝ Γ ⊢ qualified-identifier ⇓ Ϝ   Ϝ = ((Vn), (τm), τ, T, V, F)   T(identifier) = ( τ' , κ ) Γ ⊢ qualified-identifier . identifier ⇓ ( τ' , κ ) Γ ⊢ qualified-identifier ⇓ Ϝ   Ϝ = ((Vn), (τm), τ, T, V, F)   V (identifier) = τ' Γ ⊢ qualified-identifier . identifier ⇓ ∀ (Vn). (τm) → τ' Γ ⊢ qualified-identifier ⇓ Ϝ   Ϝ = ((Vn), (τm), τ, T, V, F)   F (identifier) = Ϝ' Γ ⊢ qualified-identifier . identifier ⇓ Ϝ'

Rules for atomic-kind-expression

Γ ⊢ type ⇓ type Γ ⊢ + type ⇓ +type Γ ⊢ - type ⇓ -type Γ ⊢ kind-expression ⇓ κ Γ ⊢ ( kind-expression ) ⇓ κ

Rules for multiplicative-kind-expression

Γ ⊢ atomic-kind-expression ⇓ κ Γ ⊢ atomic-kind-expression ⇓ κ Γ ⊢ atomic-kind-expression ⇓ κ1   Γ ⊢ multiplicative-kind-expression ⇓ κ2 Γ ⊢ atomic-kind-expression * multiplicative-kind-expression ⇓ ( κ1 × κ2 )

Rules for kind-expression

Γ ⊢ multiplicative-kind-expression ⇓ κ Γ ⊢ multiplicative-kind-expression ⇓ κ Γ ⊢ multiplicative-kind-expression ⇓ κ1   Γ ⊢ kind-expression ⇓ κ2 Γ ⊢ multiplicative-kind-expression -> kind-expression ⇓ ( κ1 → κ2 )
